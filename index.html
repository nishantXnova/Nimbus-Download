// Most Impactful
The flags are already set.
Here's why they work.

The Hardware Auditor already injects optimal flags on every boot. But knowing what they mean helps you understand why your game feels smoother. The two most important: -Xmx controls max heap RAM, and the GC strategy controls pause spikes.

// Nimbus injects this for a 4GB / 4-core machine
-Xms512M     // start heap small
-Xmx2048M    // max 2GB for game
-XX:+UseSerialGC // less overhead
-XX:MaxGCPauseMillis=50
-XX:-OmitStackTraceInFastThrow
// Auto-generated. You don't touch this.
🧠
Max Impact

Never exceed 75% of your RAM

Giving Minecraft more RAM than your system has free causes swapping to disk — which is 100× slower than RAM. On 4GB: keep -Xmx at 2GB max.

-Xmx2048M  ← safe for 4GB total
Impact95%
♻️
Easy

SerialGC beats G1GC on weak CPUs

G1GC is great for 8+ cores but adds overhead on 2-core machines. SerialGC is simpler, uses less CPU time, and causes fewer random lag spikes on low-end hardware.

-XX:+UseSerialGC
Impact80%
🔕
Easy

Kill background apps before launching

Chrome, Discord, Spotify each eat 200–400MB of RAM. On 4GB that's your entire Minecraft budget. Close them before launch — not after the lag starts.

Impact70%

// System Level
Windows is working against you.
Here's how to fight back.

Windows background services, power throttling, and driver overhead can eat 10–25% of your performance before Minecraft even loads. These tweaks cost nothing and take under 5 minutes.

// Windows power plan command (run as admin)
C:\> powercfg /setactive
8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
// That GUID = High Performance mode
// Instant 10–20% more CPU headroom
⚡
Max Impact

Switch to High Performance power plan

Balanced mode throttles your CPU to save battery. High Performance mode lets it run at full clock speed. On a 1.8GHz i3 this means the difference between 1.8GHz and 2.4GHz turbo.

Control Panel → Power Options → High Performance
CPU Headroom+20%
🧊
Medium

Keep your laptop cool

Thermal throttling is the silent FPS killer. When your CPU hits 90°C+ it cuts its speed in half automatically. Use a laptop stand, clean the vents, or buy a $5 cooling pad.

Throttle Prevention+30%
🛑
Easy

Disable Windows startup apps

Teams, OneDrive, Spotify, Discord — they all auto-start and sit in RAM doing nothing. Open Task Manager → Startup, and disable everything you don't need running in the background.

Ctrl+Shift+Esc → Startup tab → Disable all
Free RAM+400MB
🔍
Easy

Disable Windows Search indexing

Windows Search constantly scans your disk in the background. On slow HDDs this causes random lag spikes mid-game. Disable it during gaming sessions.

services.msc → Windows Search → Disabled
Disk Usage-40%
🎮
Medium

Set Java process priority to High

Windows gives all processes equal CPU time by default. Setting Java to High priority means the OS will prioritise Minecraft over background tasks when CPU time is scarce.

Task Manager → java.exe → Set Priority → High
CPU Priority+15%
🖥️
Easy

Update your GPU driver

Intel HD Graphics drivers from 2016 have known OpenGL bugs that cause Minecraft to run poorly. Update through Device Manager or Intel's Driver Support Assistant — free, 5 minutes.

device manager → Display adapters → Update driver
